MyTMDB_App 架構與功能說明

MyTMDB_App 是以 TMDB 內容探索為核心的 iOS App，採用 UIKit、MVVM、Service、Repository 與 Router 分層，並以 Swift Concurrency 處理 API 請求、Session 驗證與畫面狀態更新。

Deployment Target
iOS 18.4
Language
Swift 6.0
Primary UI
UIKit + UICollectionView
Architecture
MVVM / Service / Repository / Router

目前功能

App 目前涵蓋首頁推薦、電影與劇集分類列表、搜尋、Movie/TV/Season/Episode/Person 詳細頁、 會員中心、收藏、觀看清單與評分流程。

登入與 Session

  • 支援 TMDB 帳號登入與訪客 Session。
  • 登入流程為 Request Token、Validate With Login、Session 建立。
  • 啟動時驗證已儲存的使用者 Session，失效時回到登入頁。

首頁探索

  • 首頁包含 Trending、Popular、Now Playing、Upcoming、Top Rated 與 TV on air 內容。
  • MainHomeService 負責 API 請求。
  • MainHomePresentationBuilder 負責組裝畫面資料。

電影與劇集列表

  • MainMovieList 使用 Genre 與 Discover Movie API。
  • MainTVList 使用 Genre 與 Discover TV API。
  • 支援分類篩選、排序與分頁載入。

搜尋

  • MovieSearch 使用 TMDB Search Movie API。
  • TVSearch 使用 TMDB Search TV API。
  • 搜尋關鍵字會先移除前後空白，並排除 adult 內容。

詳細頁

  • 支援 Movie、TV、Season、Episode、Person 詳細頁。
  • 整合基本資訊、演員/製作人員、影片、圖片、推薦內容與觀看平台。
  • Movie/TV 共用 DetailBase 與 DetailRouter。

會員與互動

  • 會員中心支援 favorite、watchlist、rated 內容與 TMDB lists。
  • 收藏與觀看清單透過 TMDB Account API 更新。
  • 評分使用 RatingPageSheetViewController 半頁介面。

技術架構

  • View畫面組裝、使用者互動、collection view layout、ViewModel 綁定。
  • ViewModel管理 loading、loaded、empty、failed 等狀態與 presentation model。
  • Service封裝 TMDB API 呼叫、query items 與 JSON decode。
  • Repository處理跨 API 的內容組裝、快取與會員中心資料彙整。
  • Router集中處理 push、present、page sheet、Safari 等導航規則。

共用元件與主題

CineBase Logo

App 主題 logo 使用 CineBase1x.png，位於 MyTMDB_App/Assets.xcassets/AppIcon.appiconset。

ThemeColor

使用 TMDB 品牌色 #0D253F、#01B4E4，並搭配系統色支援 Dark Mode。

AppFactory

集中建立常用 Label、Button 與 Animation View，減少畫面層重複的 UI 建構程式碼。

AppAnimationView

集中處理 Lottie 動畫生命週期，目前保留 loadingAir、searchLoading、 popcornLoading 等既有動畫資產。

安裝與執行

Clone 專案

git clone https://github.com/toolman5487/MyTMDB_App
cd MyTMDB_App

使用 Xcode 開啟

開啟 MyTMDB_App.xcodeproj，並確認 Swift Package 依賴已由 Xcode 解析完成。

設定 TMDB API Key

Network/APIConfig.swift 會從 Info.plist 的 TMDBAPIKey 讀取 API Key。

<key>TMDBAPIKey</key>
<string>YOUR_TMDB_API_KEY</string>

執行 App

選擇 iOS Simulator 或真機，按下 Xcode Run。首次啟動會依 Session 狀態進入登入頁或主 Tab。

API 與在地化

NetworkService

API base URL 為 https://api.themoviedb.org/3，請求會自動補上 api_key。

會員權限

favorite、watchlist、rating 與 account list 相關 API 會額外帶入 session_id。

AppLocalization

依系統語言產生 TMDB language、region、timezone 與圖片/影片語言參數。
