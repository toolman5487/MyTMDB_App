TMDB iOS App 作品集

專案概述 專案結構 使用技術 聯絡我

一、專案概述

這是一個以 UIKit + Combine + MVVM 架構開發的 TMDB（The Movie Database）iOS 應用程式範例， 完整整合了 TMDB API，包括電影列表、電影詳情（包含預告片、演員名單）、喜好功能與評分功能， 並使用 SnapKit 做自動排版、SDWebImage 處理圖片載入、YouTube-Player-iOS-Helper 播放預告片。

以上結構明確區分：
1. Service：負責與 TMDB API 溝通。
2. Models：定義 JSON 對應的資料模型。
3. View：各畫面（Movie、TV、Person、Home 等）與客製化的 UITableViewCell。
4. ViewModel：MVVM 中介層，負責商業邏輯、資料綁定與 Combine Publisher/Subscriber。

三、使用技術

  • Swift 5.7+
  • UIKit + Combine + MVVM
  • SnapKit（Auto Layout）
  • SDWebImage（圖片快取載入）
  • YouTube-Player-iOS-Helper（播放 YouTube 預告片）
  • TMDB RESTful API
  • GCD / DispatchQueue（非同步處理）
  • CocoaPods / SPM（套件管理）

使用 MVC 已熟悉後，改採用 MVVM 架構，將View與Model解耦， 透過ViewModel處理 API 呼叫、資料轉換與錯誤處理，再以Combine做資料綁定。
